# CartNova 🛒

CartNova is a full-stack E-Commerce web application built using the MEAN stack (MongoDB, Express.js, Angular, Node.js).

## 🚀 Features

* User Authentication (JWT-based)
* Product Listing & Details
* Add to Cart & Checkout
* Order Management
* Admin Panel

## 🛠️ Tech Stack

* Frontend: Angular
* Backend: Node.js + Express.js
* Database: MongoDB
